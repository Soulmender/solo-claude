# /improve

Wraps `improvement-advisor`. Scores candidates by impact × effort, tiers them into quick-wins / strategic / long-haul, and writes `docs/improvements.md` + `improvements.json`.

```
/improve
/improve issues                 # offer to file the top items via github-issue-creator
```
